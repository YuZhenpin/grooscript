[source,groovy]

include::../src/test/resources/doc/Operators.groovy[tags=operators,indent=0]

Also you can overload this operators in your classes:

.Overload operators
|===
|operator |function

|a + b
|a.plus(b)

|a - b
|a.minus(b)

|a * b
|a.multiply(b)

|a ** b
|a.power(b)

|a / b
|a.div(b)

|a % b
|a.mod(b)

|a << b
|a.leftShift(b)

|a >> b
|a.rightShift(b)
|===
